# ajaxDemo
#Ajax 的优点: 
  <br>1.可以无需刷新页面而与服务器端进行通信<br>
  2.允许你根据用户事件来耿勋部分页面内容<br>
#Ajax 的缺点： 
  <br>1.没有浏览历史，不能回退 <br>
  2.存在跨域问题（同源）<br>
  3.SEO 不友好
